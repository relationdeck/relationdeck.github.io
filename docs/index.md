# 關係卡牌資源中心

歡迎來到 **RelationDeck 關係盒子**的卡牌資源中心！

我們精心整理了各種情境的對話卡牌問題與使用指南——朋友聚會、情侶深談、家庭時光、職場交流、陌生人破冰，以及個人反思。
每一個問題，都是一次打開真實對話、加深彼此連結的邀請。

## 卡牌情境分類

- **朋友聚會**：適合打開話匣子、分享故事，讓友情更深厚
- **情侶深談**：超越日常閒聊，探索彼此的夢想、價值觀與成長故事
- **家庭時光**：跨越世代的問題，讓家人之間多一些理解與笑聲
- **職場交流**：打破同事距離，建立更真實的信任與合作默契
- **陌生人破冰**：讓初次見面也能自然輕鬆地展開真誠對話
- **個人反思**：一個人也能和自己好好對話，探索內心世界

## 如何開始使用？

1. 從左側選單選擇你感興趣的卡牌情境
2. 找到適合當下關係的問題，開始對話
3. 收藏打動你的卡牌，記錄你的回答與感受
4. 累積對話筆記，建立屬於你的關係日記
